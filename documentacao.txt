estrategia de criação:
1- Será usado o figma para prototipar a visualização de como ficará o site

2- fazer todo HTML

3- depois do html pronto, fazer a estilização do site com CSS

4- seguir padrão de cores de uma forma moderna

cores do site:
Roxo = #8842EB
Preto = #000000
Amarelo = #FDCB06
Dica de Aplicação (Regra 60-30-10)
Para que o site não fique confuso visualmente, aplique as cores escolhidas nesta proporção:
60% (Cor Dominante): Geralmente o tom mais claro (fundo).
30% (Cor Secundária): O tom médio ou escuro (menus, rodapés, títulos).
10% (Cor de Destaque): A cor mais vibrante (apenas em botões e links importantes).

Fontes: Jaro; tamanho 48
OBS: link clicado fica amarelo e os outros permanece preto
